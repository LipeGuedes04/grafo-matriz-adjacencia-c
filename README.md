# Grafo utilizando Matriz de Adjacência

Trabalho desenvolvido para a disciplina de Estrutura de Dados Avançada.

## Sobre o projeto

Este programa implementa um grafo utilizando uma matriz de adjacência na linguagem C. A aplicação permite criar um grafo com uma quantidade definida de vértices e realizar operações básicas de manipulação e consulta.

## Funcionalidades

* Limpar o grafo
* Adicionar arestas
* Remover arestas
* Exibir a matriz de adjacência
* Calcular o grau de cada vértice
* Listar os vértices adjacentes

## Como funciona

O usuário informa a quantidade de vértices do grafo no início da execução. Depois disso, é possível utilizar o menu para adicionar ou remover conexões entre os vértices e visualizar as informações do grafo.

## Como executar

Compilar:

```bash
gcc grafo.c -o grafo
```

Executar:

```bash
./grafo
```

## Menu do programa

```text
1 - Limpar grafo
2 - Inserir/remover aresta
3 - Mostrar matriz
4 - Mostrar graus
5 - Mostrar adjacências
6 - Sair
```
